# Pretraining Datasets

Buyuk gercek pretraining corpus dosyalari bu repoya commit edilmez. Bu klasor
yalniz yerlesim sozlesmesini ve gerekiyorsa kucuk denetim orneklerini tasir.

Calisma ortamlarinda beklenen dosyalar:

```text
LaflaAI100M/data/train.jsonl
LaflaAI100M/data/veri_manifesti.json
```

Kaynak ve oran sozlesmeleri `configs/data/source-plans/` altindadir.
